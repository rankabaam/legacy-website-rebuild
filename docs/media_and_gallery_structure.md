# Media and Gallery Structure

The website needed a practical structure for recurring media updates such as sermon videos, photos, and event content.

## Media Requirements

- Sermon video access
- Photo gallery organization
- Event-related media updates
- Simple content update process
- Public-facing presentation suitable for church visitors

## Gallery Considerations

The gallery structure needed to support ongoing updates without requiring a full redesign each time new photos were added.

## Sermon Media Considerations

The sermon media section was designed to support recurring video updates. This connects naturally with a separate sermon video automation workflow, where raw recordings can be processed into final video, thumbnail, and audio outputs before being uploaded or linked from the website.

This creates a practical end-to-end workflow:

```text
Raw sermon recording
→ Video processing and thumbnail creation
→ Final upload-ready media files
→ Website sermon page or archive update
