# Translation Integration

The rebuilt website later supported access to live translation resources through a dedicated translation link.

## Purpose

The live translation access was designed to support multilingual church attendees during worship services.

## Integration Concept

The main website provides a clear access point for visitors who need live translation support. The translation system itself is treated as a separate service, while the website acts as the public entry point.

## Design Considerations

- Keep the main website stable
- Keep live translation access easy to find
- Avoid mixing experimental translation system logic into the main website
- Allow the translation system to be updated independently
