# Rebuild Goals

The goal of the rebuild was not simply to recreate the old website, but to create a more maintainable structure for a real organization with recurring content needs.

## Main Goals

- Replace the unreliable legacy CMS workflow
- Rebuild the site using WordPress
- Improve menu and page organization
- Make photo and media updates easier to manage
- Support sermon and service-related content
- Create a structure that can support future integrations
- Keep the website maintainable for non-technical users

## Design Priorities

- Clear navigation
- Simple public-facing structure
- Maintainable content workflow
- Stable media organization
- Mobile-friendly presentation
- Practical long-term management
