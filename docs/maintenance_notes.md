# Maintenance Notes

The rebuild focused on creating a website structure that could be maintained more easily over time.

## Maintenance Priorities

- Avoid dependence on an unsupported legacy CMS
- Use a more common website management platform
- Keep menus and pages understandable
- Organize media content in a repeatable way
- Separate public content from experimental or operational tools

## Privacy and Security

This portfolio repository does not include:

- Administrator credentials
- Hosting account information
- Server configuration details
- Private church records
- Original media files
- Internal-only documentation
